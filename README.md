
# Blog

A simple bloging website..



## Tech Stack


**Client:** React , TailwindCSS

**Server:** hono, noserver


**DB:** postgress, Prisma




## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start React dev

```bash
  npm install
```

Start the server

```bash
  npm run start
```
